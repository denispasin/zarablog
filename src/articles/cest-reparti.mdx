J'ai eu besoin d'un médium pour partager des vidéos que je fais pour [THP](https://www.thehackingproject.org) et du coup me "_revoilà_".
J'ai supprimé tous les articles non finis/à demi commencés qui trainaient dans ce repo perdu (même pas sur github :D).

En attendant que voit le jour un futur side-project pour présenter un cours de manière agréable et pratique (#teaser) je vais m'en servir pour les présenter d'une manière plus agréable que directement sur Dropbox.

<!--more-->

## Changement d'hebergement/source-control

Gosh… C'était stocké uniquement sur mon PC perso. Toute cette config et ces posts n'était qu'a un SSD crash de l'extinction.
Ça va donc partir sur Github et en hébergement et en source-control

## Changement de pipeline

Jenkins c'est cool mais j'ai clairement autre chose à faire.
Donc direction [circleci](circleci.com) qui a chaque commit sur Github va se faire un plaisir de build le site.
